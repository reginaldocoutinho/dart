# Primeiro Projeto em Dart

# Lista de comandos
 *executar o programa*
 - dart run

 *executar passando argumentos*
 - dart bin/cli.dart help
 - dart bin/cli.dart version *mostra a versão configurada no app*
 - dart bin/cli.dart search
 - dart bin/cli.dart search Dart Programming *enviando um article title*
 - dart pub get *este comando serve para sincronizar as novas dependências inseridas*
